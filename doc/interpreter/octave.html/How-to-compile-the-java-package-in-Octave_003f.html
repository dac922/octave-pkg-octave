37.3.5 How to compile the java package in Octave?

Most Octave installations come with the java package pre-installed. In case you want to replace this package with a more recent version, you must perform the following steps:

  • Uninstall the currently installed package java
  • Make sure that the build environment is configured properly
  • Compile and install the package in Octave
  • Test the java package installation
